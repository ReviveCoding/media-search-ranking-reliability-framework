# Security Policy

## Supported version

The latest commit on `main` is the supported version of this portfolio repository.

## Reporting a vulnerability

Do not open a public issue containing secrets, credentials, exploit details, or sensitive data.

Use GitHub private vulnerability reporting when enabled. Otherwise, contact the maintainer through the GitHub profile associated with this repository and provide a minimal description without publishing exploit details.

## Scope

This repository contains research and portfolio code. It is not represented as a production service and does not process proprietary user data by default.
